# 开发计划

## 第一阶段：MVP

- 创建课程
- 上传并解析 PDF
- 文本切分与 ChromaDB 索引
- 基于课程资料的问答
- 返回来源引用

## 第二阶段：学习功能增强

- 自动总结课件
- 生成复习题
- 多语言术语解释
- 保存聊天记录

## 第三阶段：产品完善

- 文档处理状态改为后台任务
- 支持流式模型输出
- 导出总结和复习题
- 为 README 补充截图
- 增加 GitHub Actions CI
- 强化科技风 UI：动态网格、HUD 信息卡片、发光边框与高级微交互
- 完善多页面结构：首页、课程库、详情页、AI 工具台、学习洞察页

## 第四阶段：高级功能

- 支持 PPTX 和 DOCX
- 支持扫描版 PDF OCR
- 增加记忆卡片和间隔复习
- 增加错题本
- 增加知识图谱可视化
- 增加云端部署配置

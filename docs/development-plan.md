# 开发计划

## 第一阶段：MVP 闭环

- 创建课程
- 上传并解析 PDF
- 文本切分并写入 ChromaDB / 本地向量索引
- 基于课程资料问答
- 返回来源文件和页码引用

## 第二阶段：产品化增强

- 多 PDF 批量上传，统一加入课程知识库
- RAG 混合检索：语义向量 + 关键词重排 + 多文档去重
- 课程详情页升级为学习工作台
- 结构化 Quiz 生成、作答、自动判分和解析反馈
- 用户级学习统计：课程数、文档数、题库数、平均得分、近期活动
- 修复中文文案编码，提升页面可读性

## 第三阶段：学习闭环深化

- 保存更完整的聊天历史和学习路径
- 错题本与薄弱知识点自动归纳
- 记忆卡片和间隔复习
- 总结与 Quiz 导出为 PDF / Word
- 流式模型输出
- 文档处理改为后台任务队列

## 第四阶段：高级能力

- 支持 PPTX 和 DOCX 解析
- 支持扫描版 PDF OCR
- 支持知识图谱可视化
- 增加教师端和班级空间
- 增加云端部署配置、CI/CD 和生产安全策略
